# rest-basic
| ROUTE          | HTTP VERB | DESCRIPTION      |
|----------------|-----------|------------------|
| /API/data      | GET       | get all data     |
| /API/data      | POST      | create a data    |
| /API/data/:id  | GET       | get a data       |
| /API/data/:id  | PUT       | update a data    |
| /API/data/:id  | DELETE    | delete a data    |
